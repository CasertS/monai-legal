Privacy Policy — MonAI

Ultimo aggiornamento: giugno 2026
Sviluppatore: Team MonAI — laneup.fit@gmail.com

Cosa fa MonAI

MonAI è un'app di gestione delle spese personali con assistente IA. Tutti i tuoi dati finanziari (conti, transazioni, categorie, budget, investimenti) sono salvati esclusivamente sul tuo dispositivo e non vengono mai trasmessi a server di MonAI né condivisi con terzi.

Dati raccolti

MonAI non raccoglie né trasmette dati personali propri. L'app non ha un backend, non richiede registrazione, non crea account.

Intelligenza Artificiale

Se scegli di usare un provider IA (Google Gemini, Anthropic Claude, OpenAI GPT), il testo che inserisci nella chat viene inviato direttamente al provider che hai selezionato, usando la tua chiave API personale. MonAI non vede né registra questo traffico. Le chiavi API sono salvate nel Keychain/Keystore del tuo dispositivo e non lasciano mai il dispositivo stesso.

Privacy policy dei provider:
• Google Gemini
• Anthropic Claude
• OpenAI GPT

Dati biometrici

Se attivi il blocco con Face ID/biometria, MonAI usa le API native del sistema operativo. MonAI non ha accesso ai dati biometrici, che restano nel Secure Enclave del dispositivo.

Backup

I backup che esporti (JSON, CSV, Excel) vengono salvati nella memoria locale del dispositivo o condivisi tramite le app che scegli tu. MonAI non carica backup su propri server.

Notifiche

Le notifiche sono generate localmente dal dispositivo. Non vengono usati servizi di push notification di terze parti.

Contatti
